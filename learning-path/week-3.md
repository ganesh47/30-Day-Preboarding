# Week 3 — Structuring, Covenants, and Monitoring

## Theme
Design protections that work before distress becomes irreversible.

## Objectives
- Build covenant logic tied to real risk triggers
- Understand collateral and enforceability trade-offs
- Translate warning signals into action playbooks

## Daily focus areas
- Term sheet architecture
- Covenant and trigger calibration
- Security package enforceability
- Watchlist and early warning workflows
- Committee simulation on stressed borrower

## Output artifact
Term sheet with rationale for each protection.

## Reflection prompts
- Which protections are cosmetic vs decisive?
- How fast can the system react after first stress signal?

## Executive summary task
Write: "Three protections that materially improve downside outcomes."

## Tech leverage task
Define alert-to-action workflow with owners and SLAs.

## Operator instinct task
Re-negotiate one covenant under commercial pressure.
