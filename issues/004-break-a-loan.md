# Issue 004 — Break a Loan

## Objective
Stress-test a proposed loan and identify how it can fail.

## Tasks
- Define base case assumptions behind approval.
- Build three break scenarios (margin shock, working capital shock, demand shock).
- Quantify covenant headroom erosion in each scenario.

## Mini deliverable
Stress note with scenario outcomes and recommended protections.

## Reflection question
Which assumption is carrying most of the approval confidence?

## Optional stretch goal
Propose alternate repayment schedule more aligned to downside cash flow.

## Checklist
- [ ] Base case documented
- [ ] Three stress scenarios completed
- [ ] Protection recommendations drafted
- [ ] Reflection logged
