# Issue 006 — Design Covenants

## Objective
Design a covenant package that enables early intervention.

## Tasks
- Propose financial and operational covenants.
- For each covenant: threshold, frequency, data source, and escalation action.
- Separate cosmetic covenants from actionable ones.

## Mini deliverable
Covenant matrix with trigger-action mapping.

## Reflection question
Which covenant would I defend most strongly in negotiation, and why?

## Optional stretch goal
Simulate one covenant breach and draft response playbook.

## Checklist
- [ ] Covenant matrix completed
- [ ] Trigger actions defined
- [ ] Negotiation priority noted
- [ ] Reflection logged
