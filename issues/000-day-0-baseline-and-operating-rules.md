# Issue 000 — Day 0 Baseline and Operating Rules

## Objective
Set up the operating baseline for the 30-day sprint so daily execution is measurable, consistent, and decision-focused.

## Tasks
- Write a Day 0 baseline note covering current strengths, weak spots, and assumptions about lending.
- Define success metrics for the next 30 days (e.g., issue completion rate, quality of artifacts, confidence trend).
- Set personal operating rules for cadence: daily time block, artifact standard, and end-of-day reflection discipline.
- Create a risk question backlog with at least 10 high-priority questions to resolve over the program.

## Mini deliverable
A one-page "Day 0 Brief" with baseline assessment, metrics, operating rules, and top risk questions.

## Reflection question
What would make this month look "busy" but still fail to improve real credit judgment?

## Optional stretch goal
Add a scoring rubric to self-grade each daily artifact for clarity, rigor, and decision usefulness.

## Checklist
- [ ] Baseline note written
- [ ] 30-day success metrics defined
- [ ] Operating rules documented
- [ ] Risk question backlog created
- [ ] Reflection logged
