# Week 1 — Lending Machine Fundamentals

## Theme
How value is created or destroyed across origination to repayment.

## Objectives
- Map end-to-end lending workflow and control points
- Understand primary drivers of repayment and loss
- Build first-principles risk vocabulary

## Daily focus areas
- Day 1: Map lending value chain
- Day 2: Economics of repayment vs disbursal
- Day 3: Balance sheet and cash flow basics
- Day 4: Loan structure mechanics
- Day 5: Collections reality

## Output artifact
A one-page operating map with failure points and leading indicators.

## Reflection prompts
- Where can a "good" loan still become a bad outcome?
- Which process stage has the weakest feedback loop?

## Executive summary task
Write a 300-word note: "What actually drives lending outcomes?"

## Tech leverage task
Design a minimal risk dashboard schema with 10 fields.

## Operator instinct task
Pick one stage and define escalation thresholds.
