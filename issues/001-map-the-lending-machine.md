# Issue 001 — Map the Lending Machine

## Objective
Build a clear end-to-end map of the lending operating system from sourcing to recovery.

## Tasks
- Document stages: origination, underwriting, committee, documentation, disbursal, monitoring, collections, recovery.
- For each stage, define owner, key decision, primary risk, and control mechanism.
- Mark where information is lost between teams.

## Mini deliverable
A one-page lifecycle map with top 10 failure points.

## Reflection question
Where can a portfolio look healthy while hidden risk is accumulating?

## Optional stretch goal
Add leading indicators for each failure point.

## Checklist
- [ ] Lifecycle map completed
- [ ] Owners and controls documented
- [ ] Failure points prioritized
- [ ] Reflection logged
